## Core Formatting ğŸ¨

These are the fundamental tools for styling your text and structuring your document.

### Text Styles

- **Basic Formatting:** Apply **Bold**, _Italic_, <ins>Underline</ins>, and ~~Strikethrough~~.
- **Inline Code:** Use the `Code` format for monospaced text.
- **Color:** Change the text color and background (highlight) color.
- **Advanced:** Apply Subscript, <sup>Superscript</sup>, and change text case (UPPERCASE, lowercase, Capitalize).
- **Special Text:** Wrap text in `[square brackets]` to automatically make it **bold** and highlight it in yellow.

### Paragraph Styles

- **Headings:** Organize your content with three levels of headings (H1, H2, H3).
- **Blockquote:** Create an indented block of text, perfect for quoting sources.
- **Lists:** Create bulleted, numbered, or clickable checklist-style lists.
- **Alert Boxes:** Add colored alert boxes for Info, Success, Warning, Error, or Urgent messages.
- **Code Blocks:** A dedicated block for displaying multi-line code, with syntax highlighting for various languages.

### Alignment & Indentation

- Align text to the **Left**, **Center**, **Right**, or **Justify** it.
- Increase or decrease the indentation of any text block.

## Productivity & Workflow ğŸš€

The editor includes several features designed to make your writing process faster and more efficient.

### Input Methods

- **Slash Commands:** On a new line, type `/` to open a powerful command menu. From here, you can search for and insert almost any element. For example, `/h1` for a heading or `/table` for a table.
- **Markdown Auto-formatting:** Use familiar Markdown syntax to format on the fly (e.g., `# ` for an H1, `* ` for a list, `---` for a divider).

### Keyboard Shortcuts âŒ¨ï¸

Nearly every action has a keyboard shortcut. For example:

- `Ctrl+B` for **Bold**
- `Ctrl+1` for **Heading 1**
- `Ctrl+Shift+*` for a **Bulleted List**
- ...and many more! Hover over any button in the toolbar to see its shortcut.

### Content Interaction

- **Floating Toolbar:** When you select a piece of text, a small formatting toolbar appears directly above it for quick access to common actions like bold, italic, and underline.
- **Drag & Drop Blocks:** Hover over any paragraph, heading, or image to reveal a drag handle. Click and drag this handle to easily reorder blocks of content âœ¥.
- **Custom Context Menu:** Right-click to open a context menu with actions like Copy, Paste, and Delete. The menu is contextual, showing a "Remove Link" option when you right-click on a link, for example. ğŸ–±ï¸

### Content Creation Tools

- **Mentions:** Type `@` followed by a name to bring up a list of users and mention them in the document.
- **Emojis:** ğŸ˜„
  - **Picker:** Type `:` followed by a name (e.g., `:smile`) to open a searchable emoji picker.
  - **Auto-conversion:** Common emoticons like `:)` are automatically converted into emojis.
- **Speech-to-Text:** Click the microphone icon to enable dictation and transcribe your voice directly into text ğŸ¤.
